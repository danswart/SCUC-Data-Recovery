# Incoming TEA updates

Place newly downloaded Texas Education Agency source files here before changing
any canonical SCUC Snapshot dataset.

Keep the original download unchanged. Use a dated subfolder for each update,
for example `2026-07-23 SAT update`, and record the source URL and download date.
Working files derived from that source may remain in the same dated subfolder
until the candidate dataset has passed the checks in
`SCUC_CANONICAL_DATA_MAINTENANCE_GUIDE.qmd`.

Nothing in this folder is canonical. Analytical projects must read approved
files from `01 reference canonical`.
